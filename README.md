# About Me
Full Stack Developer | AI Systems | Simulation Engineering

I am a self-taught developer based in Mumbai, currently pursuing a B.Tech in Computer Science and Engineering (Data Science). My work focuses on building scalable systems, real-time simulations, and intelligent architectures that combine software engineering with physics-inspired modeling.

I am particularly interested in developing high-performance environments, digital twin systems, and AI-driven applications that go beyond traditional web development.

## Vision

I aim to build advanced computational systems that bridge real-world physics, artificial intelligence, and large-scale simulation.

My long-term focus includes:
- Real-time digital twin environments
- Physics-based simulation systems (space, urban environments)
- AI-driven autonomous systems and memory architectures
- High-performance, low-level optimized software
